Frequently asked questions

Can I move text and signatures after placing them?

Yes. Use the Select tool to drag text, images, and signatures. Corner handles resize them; Lock prevents accidental edits. Double-click text to edit the wording.

Can I edit existing PDF text?

This editor adds annotations on top of pages (text, draw, highlight, redact). Full reflow editing of native PDF text isn’t available client-side.

Do I need an account?

No — never.
