Frequently asked questions

How do I password protect my Aadhaar or PAN card PDF?

Upload the PDF, type a password twice to confirm, and click Protect PDF. The downloaded file will ask for that password whenever anyone tries to open it — useful before emailing ID documents.

What encryption does ToolShoppy use?

AES-128, the PDF standard security handler (the same base scheme most PDF readers and free tools use). It's suitable for keeping casual viewers and automated scraping out of a document — not a substitute for a full digital signature or bank-grade security workflow.

Is my document uploaded anywhere?

No. The PDF is parsed, encrypted, and re-downloaded entirely inside your browser using the Web Crypto API. It never leaves your device — which matters a lot for ID documents like Aadhaar, PAN, and Voter ID.

What if I forget the password?

There is no recovery option — ToolShoppy does not store your password anywhere. Write it down somewhere safe before sharing the file.

Can I protect a PDF that's already password protected?

Not directly — you'd need to remove the existing password first (in the app that created it, or a PDF unlock tool) before adding a new one here.
