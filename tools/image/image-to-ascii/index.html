Where this works best

          
    
            
  • Width is capped at ~26 characters by default on purpose. A row that fits fine in a wide PC browser window will silently wrap onto the next line in a narrow phone chat bubble — and one wrapped row shifts every row below it, turning the whole image into noise. This has been tested on real narrow screens; going wider risks exactly that distortion on mobile.
    • 
            
  • Instagram / Facebook / LinkedIn comments — use Braille. These platforms strip monospace fonts and collapse spaces, but Braille dot-patterns stay recognisable in any font. Each has its own comment character limit — use the "Posting to" chips above so the art is auto-sized to fit in one paste.
    • 
            
  • WhatsApp / Telegram chat — wrap the copied text in triple backticks (```like this```) when you paste, to force a true monospace font so rows line up perfectly. Pasting without the backticks uses the app's normal font, which is not guaranteed to line up.
    • 
            
  • Check the mobile preview below the art before posting — it's rendered at real phone-bubble width, so if it's going to wrap, you'll see it there first.
    • 
            
  • Outline only traces just the edges instead of shading the whole photo — it's the fastest to recognise and reads clearly at an even smaller width, so it fits in a shorter comment.
    • 
            
  • Anywhere as a picture — hit "Download as Image" for a PNG sized to that platform's real canvas (1080×1080 for feed/comment images, 1080×1920 for WhatsApp Status) so it fills the frame properly on a phone instead of getting letterboxed or cropped.

Why text art beats a plain photo in comments

    
Instagram, Facebook and LinkedIn comment sections don't let you attach a photo the way a post does — but they always accept plain text. Converting your product shot, logo or face into Braille or block text art turns an ordinary comment into something people stop and read, which is exactly why brands and creators use it for cheeky call-to-action comments, giveaway teasers and "look closer" style engagement bait. Because everything runs locally in your browser using the Canvas API, no image is ever uploaded anywhere — pick a style, tune the width, then copy or download and post.
